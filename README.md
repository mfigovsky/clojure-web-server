# Clojure Web Server

A demo of a Catacumba web server buit with components.

## Installation

~~~bash
lein deps
~~~

## Run locally

~~~bash
lein run -m clojure-web-server.core
~~~

## Run locally inside a REPL
~~~bash
lein repl
user => (go)
~~~

## Run as a jar

~~~bash
lein uberjar
java -jar target/uberjar/clojure-web-server-standalone.jar
~~~
